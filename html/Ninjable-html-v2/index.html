FAQ

Does it have all the access to my data?

Every company has their own hosted separately environment, so all the data controlled byCompany, we are as a provider only have access to encrypted(no personal data IDs) connected topositions, skills and speed of development, so we can analyze common data, like most used skills, time for level upgrade etc.

Can it be implemented for one team and then be scaled to the whole company?

Yes, it can. NINJABLE has a department structure that allows you to add any additional departments and people later without interruption to previously created departments and their progress.

Can it be customized?

Yes, additional customization can be provided byrequest to our team. Some of the customizations can require additional one-time payment fordevelopment.
